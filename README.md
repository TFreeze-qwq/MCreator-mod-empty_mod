# MCreator-mod-empty_mod
MCreator制作的Minecraft 1.20.1 Forge模组\
共添加了两个指令
# fkill
``/fkill <entities>``\
强制清除实体，无尽套防不住（更强的kill说是）
# protect
``/protect <entities> <true or false>``\
开关生物保护，免疫**fkill kill**\
死亡后**保护失效**
# debugprotectforcekill
**仅限debug版**，可以直接杀死受保护的生物\
用法与fkill一致
## **以上指令均需要4级权限**
(模组名称与模组内容没有任何直接关系，只是不会起名字罢了qwq)
